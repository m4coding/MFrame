(2)FragmentPagerAdapter和FragmentStatePagerAdapter的区别
FragmentPagerAdapter切换的时候会保存fragment的状态，用于快速恢复，用的是detach分离，fragment不会走onDestroy；
FragmentStatePagerAdapter切换的时候不会保存fragment的状态，直接remove掉，适合多个加载
